# 📊Statistical Analysis in Excel

## Project Overview

This project demonstrates how advanced Excel functions can be used to perform statistical and business analysis on a sales dataset containing over **1,000 transaction records**. The objective was to answer multiple business-oriented analytical questions without relying solely on Pivot Tables, showcasing formula-based problem-solving and data analysis skills.

The dataset includes information such as customer details, city, state, region, product category, product name, quantity sold, price per unit, and sales amount.

---

## 🎯 Objectives

The project focuses on answering the following analytical questions:

1. Calculate the **total number of transactions for each state**.
2. Calculate the **total number of transactions by state and product category**.
3. Calculate the **total quantity sold by region**.
4. Calculate the **total quantity sold by region and product name**.
5. Calculate the **average sales amount for each state**.
6. Calculate the **average sales amount by state and product name**.
7. Calculate the **maximum sales amount achieved in each city**.
